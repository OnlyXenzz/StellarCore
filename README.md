## 📖 ᴅᴇꜱᴄʀɪᴘᴛɪᴏɴ
StellarCore adalah plugin economy lengkap untuk Minecraft dengan sistem:

- ✅ Balance & Transfer
- ✅ Daily Reward
- ✅ Top Leaderboard
- ✅ Job System (7 pekerjaan berbeda)
- ✅ Dynamic Market (harga naik/turun)
- ✅ Stock Market (investasi saham)
- ✅ Loan System (pinjam uang)
- ✅ Tax System (pajak transaksi)
- ✅ Interest System (bunga bank)

## 📥 ɪɴꜱᴛᴀʟʟᴀᴛɪᴏɴ
1. Download `StellarCore.jar` dari [Releases](https://github.com/StellarProject/StellarCore/releases)
2. Copy ke folder `plugins/` server Minecraft
3. Restart server
4. Konfigurasi di `plugins/StellarCore/config.yml`

## 🎮 ᴄᴏᴍᴍᴀɴᴅꜱ
| Command | Description |
|---------|-------------|
| `/balance` | Cek balance kamu |
| `/pay <player> <amount>` | Kirim uang ke player lain |
| `/top` | Lihat 10 player terkaya |
| `/daily` | Klaim reward harian |
| `/job <name>` | Pilih atau lihat pekerjaan |
| `/market buy/sell` | Beli/jual item di market |
| `/stock buy/sell` | Investasi saham |
| `/loan take/repay` | Pinjam atau bayar utang |
